Múltiples entornos de evaluación

Con UNCode puedes trabajar con múltiples lenguajes de programación como Python, Java, C y C++. También, utiliza lenguajes de descripción de hardware como VHDL y Verilog. Además, puedes desarrollar e interactúar con actividades propuestas en notebooks de Jupyter.

Sistema de calificación automática

UNCode posee un sistema de calificación automática basada en INGInious por tanto es flexible, seguro, escalable y seguro. Adicionalmente, como juez está diseñado para ser aplicado a entornos de aprendizaje en lugar de competiciones de programación.

Herramientas para los estudiantes

UNCode apoya el proceso de aprendizaje de los estudiantes mediante la evaluación automática con retroalimentación inmediata. La realimentación es sumativa y formativa. La parte sumativa se le informa a los estudiantes por medio de una calificación numérica. La realimentación formativa abarca varios aspectos de la programación, donde destaca la sintaxis, semántica, mantenimiento de código y realización de pruebas personalizadas. Para esto cuenta con un "linter" para hacer recomendaciones de buenas prácticas, visualizador de código en ejecución en Python tutor (link to python tutor), entorno de pruebas de entrada y salida personalizables, plantillas de código para tareas de programación, entre otras.

Herramientas para los profesores

UNCode cuenta con un asistente de calificación, banco de problemas de programación, un módulo de evaluación manual, generador de reportes anti plagio, los cuales apoyan la labor docente.

Herramientas de monitoreo

UNCode cuenta con un módulo generador de reportes estadísticos interactivos para estudiantes y profesores.
