Multiples lenguajes

UNCode puede evaluar código fuente escrito en Java, C/C++ o Python

Sistema de calificación automática

UNCode posee un sistema de calificación automática basada en INGInious por tanto es flexible, seguro, escalable y seguro. Adicionalmente, como juez está diseñado para ser aplicado a entornos de aprendizaje en lugar de concursos de programación competitiva.

Herramientas para los estudiantes

UNCode apoya el proceso de aprendizaje de los estudiantes mediante la retroalimentación automática y formativa de varios aspectos de la programación, donde destaca la sintaxis, semántica, mantenimiento de código y realización de pruebas. Para esto cuenta con un "linter" para hacer recomendaciones de buenas prácticas, visualizador de código en ejecución, entorno de pruebas de entrada y salida personalizables, y plantillas de código para tareas de programación.

Herramientas para los profesores

UNCode cuenta con un asistente de calificación, banco de problemas de programación, un modulo de evaluación manual, generador de reportes anti plagio y visualizador de espacio de soluciones en forma de grafo, las cuales apoyan la labor de docencia.

Herramientas de monitoreo

UNCode cuenta con un modulo generador de reportes estadísticos interactivos para estudiantes y profesores.
