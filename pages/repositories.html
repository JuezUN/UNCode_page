INGInious

Es el repositorio principal de UNCode.

UNCode es una plataforma online —desarrollada sobre INGIniois (v0.5)— para practicar programación y realizar evaluación automática de código escrito por estudiantes, de forma que apoye su proceso de aprendizaje.

Actualmente la plataforma es usada en la Universidad Nacional de Colombia, sede Bogotá, para evaluar código escrito en varios lenguajes de programación como C/C++, Java y Python 3; además el juez puede ser usado en lenguajes descriptivos como VHDL y Verilog, o en ejercicios planteados en Notebooks de Jupyter. Es por esta razón que UNCode ha sido implementada en varios cursos de la universidad tales como: programación básica de computadores, estructura de datos, Machine Learning, Inteligencia artificial, programación cuántica, lenguajes de programación, entre otros.

UNCode está constituido por un backend que maneja la interacción con Docker y el evaluador, y un frontend donde los estudiantes pueden subir sus códigos en una simple pero agradable interfaz. El frontend además incluye una interfaz de administrador que permite a los profesores y tutores revisar el progreso de sus estudiantes y modificar los ejercicios de forma sencilla.

Ambas partes que constituyen UNCode, backend y frontend, son independientes entre sí. Particularmente el backend ha sido creado para poder ser usado como si fuera una librería.

Otra característica interesante de la plataforma es que puede ser implementado como un evaluador externo para LMS (Learning Managment Systems), como Moodle u OpenEDX.

INGInious Container

Este repositorio contiene los contenedores utilizados para la ejecución y evaluación de código de UNCode disponibles en Docker Hub.

No todos los contenedores son usados en UNCode, ya que parte de estos se destina para INGInious-containers. Solo están explicados los contenedores desarrollados por nosotros.

Repositorio

Python tutor

Python Tutor es una herramienta que ayuda a superar una de las mayores barreras en el proceso de aprendizaje de la programación: comprender lo que está sucediendo en cada línea del código fuente. La herramienta permite visualizar paso a paso el comportamiento de código escrito en Python, Java, JavaScript, TypeScript, Ruby, C o C++ en navegadores web.

En el repositorio se hicieron algunas modificaciones al código original con el propósito de que este componente fuera un servicio de Docker para poder ser embebido en la interfaz de UNCode, y así permitir a los usuarios visualizar código escrito en Python 3, C/C++ y Java.

El proyecto de Python tutor fue creada por Philip Guo en enero del 2010. Aquí puedes ver la historia del proyecto.

Repositorio

Linter web service

Este repositorio contiene el código encargado de ofrecer el servicio de “linting” en UNCode, mediante un contenedor que está en Docker Hub en el repositorio unjudge/linter-web-service. Este servicio ayuda a los estudiantes señalando los errores de programación o de malas prácticas.

Repositorio

Coala bears

Cola-bears es un paquete de Python que contiene todos los “bears” (o plugins) soportados oficialmente por coala. Se cuenta con más de 78 “bears” que cubren 54 lenguajes. Coala permite la creación de reglas y estándares a ser seguidos en el desarrollo de código fuente, de manera que puede hacer un proceso de “linting” (detección y marcado de errores de programación) y arreglos en el código basándose en esos parámetros. Aquí hay una lista general que contiene información sobre cada “bear”, tales como los lenguajes que soporta y las correcciones que puede aplicar al código.

Repositorio

Testing

En este repositorio se puede encontrar diferentes pruebas para aplicar a UNCode.

Dentro de las pruebas tenemos load testing. Esta prueba consiste en ver y medir el comportamiento del servidor cuando muchos usuarios están tratando de usar la aplicación al mismo tiempo. Esto puede incluir iniciar sesión, ir a la pagina de tareas y realizar un envío. Esta prueba está hecha con JMeter, probando varios flujos de trabajo como el desempeño o el envío de Notebooks.

Repositorio

Opt cpp backend

Este repositorio contiene el backend de C/C++ para Online Python Tutor. Hemos realizado un fork a este repositorio para añadir algunas modificaciones necesarias para nuestros propósitos, y así proporcionar correctamente el servicio de “Python tutor”.

El código del repositorio fue desarrollado usando Docker stored en Doker Hub: unjudge/opt-cpp-backend, que al mismo tiempo es usado por un servicio de Linux llamado cokap.
