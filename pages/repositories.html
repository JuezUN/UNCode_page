INGInious

This is the main UNCode repository

UNCode is an online platform - developed on INGIniois (v0.5)— - to practice programming exercises and perform automatic grading of code made by the students to help them in their learning processes.

UNCode is actually used by the Universidad Nacional de Colombia, Bogotá headquarters, to evaluate the written code by the students in multiple programming languages as C / C ++, Java and Python 3; also, the UNCode judge can be used in hardware description languages such as VHDL and Verilog, or with Jupyter Notebooks tasks. For this reason, UNCode has been used in different university courses as: basic computer programming, data structures, machine learning, programming languages, quantum programming, etc.

UNCode have a backend that handles the interaction with Docker and the grader, and a frontend where the students can upload their codes in a simple and pleasant interface. Also, the frontend includes an administrator interface to allows teachers and tutors check their students progress and modify the tasks on a simple way.

Both backend and frontend are independent of each other. The backend, in particular, has been created to be used as if it was a library.

Another interesting feature of UNCode is that can be used as an external grader for Learning Management Systems (LMS), as Moodle or OpenEDX.

Python tutor

Python tutor is a tool that helps the students to pass one of the most important barriers on the programming learning processes: Understand what is happening in each code line. This tool allows the student to visualize step by step, the code behavior in the different languages such as Python, Java, JavaScript, TypeScript, Ruby, C or C++ in web browsers.

Some modifications to the original code were made in this repository, with the purpose of making this component as a Docker service in order to be embedded in the UNCode interface, and then allows users to visualize code on Python 3, Java, C and C++.

The Python Tutor project was created by Philip Guo in January 2010. You can see the project history.

Coala bears

Cola-bears is a Python package that contains all the "bears" (or plugins) officially supported by coala. There are more than 78 "bears" that cover 54 languages. Coala allows the creation of standards and rules to be followed in the code development, which allows to do a "linting" process (detect and marking programming errors), and some fixes to the code acording to the source code with these parameters. There is a general list that contains information about each "bear", such as the supported languages, and the fixes that can be made to the code.
