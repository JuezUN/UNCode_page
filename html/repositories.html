INGInious

Es el juez encargado de evaluar automáticamente el código escrito por los estudiantes. Está implementado en Python, utiliza Docker para ejecutar los códigos en un ambiente seguro y mongoDB para realizar persistencia a los datos.

INGInious consta de dos partes independientes: frontend y backend. El frontend es la parte con la cual los usuarios interactúan: los estudiantes pueden subir su código en una sencilla interfaz, mientras que los profesores pueden revisar el progreso de sus estudiantes y modificar las actividades o ejercicios de manera sencilla. Por su lado, el backent, es el componente encargado de administrar la interacción con Docker y evaluar los códigos ingresados por los estudiantes.

Online python tutor

Python Tutor es una herramienta desarrollada por Philip Guo que permite visualizar el comportamiento de código escrito en Python, Java, JavaScript, TypeScript, Ruby, C o C++ en navegadores web, permitiendo que los usuarios pueden comprender que se está ejecutando paso a paso.

Linter web service

Este repositorio contiene el código encargado de ofrecer el servicio de “linting” en la plataforma, lo que significa que es una herramienta que realiza un proceso de análisis de código para señalar errores de programación.

Repositorio

Minify static

Contiene un programa realizado en Node JS que permite minificar archivos estáticos de JS y CSS en complementos en el repositorio de INGInious.

Repositorio

Coala bears

Cola-bears es un paquete de Python que contiene todos los “bears” (plugins) soportados oficialmente por coala. Se cuenta con más de 78 “bears” que cubren 54 idiomas. Coala permite la creación de reglas y estándares a ser seguidos en el desarrollo de código fuente, de manera que puede hacer un proceso de “linting” (detección y marcado de errores de programación) y arreglos en el código basándose en esos parámetros.

CodeMirror
